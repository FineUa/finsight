# Finsight — сайт порівняння фінансових продуктів

## Що це за проект
Веб-сервіс для порівняння фінансових продуктів України:
кредити, депозити, страховки. Мова інтерфейсу — українська.
Аудиторія — звичайні користувачі без фінансових знань.

## Технічний стек
- Next.js 15, TypeScript, Tailwind CSS, App Router
- Дані: mock JSON файли у папці data/
- Стилі: тільки Tailwind, без окремих CSS файлів
- Компоненти: папка components/

## Правила розробки
- Коментарі в коді — англійською
- Назви компонентів — PascalCase (ProductCard, FilterPanel)
- Назви файлів — kebab-case (product-card.tsx)
- Завжди використовуй TypeScript типи
- Після змін перевіряй npm run dev

## Структура проекту
app/          — сторінки
components/   — UI компоненти
data/         — mock дані JSON
types/        — TypeScript типи
lib/          — допоміжні функції

## Поточний статус
Базовий Next.js проект створено. Починаємо розробку з нуля.
